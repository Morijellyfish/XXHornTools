# 狩りピTools

モンスターハンターXXのプレイをサポートする各種ツール集です。

## 概要

狩りピToolsは、モンスターハンターXXの狩猟笛を中心とした攻略ツール集です。武器の性能比較やバフ管理を効率的に行えるツールを提供しています。

### 利用可能なツール

#### 狩猟笛比較表

- **詳細なバフ設定**: 攻撃力や会心率に影響する様々なバフ・スキルを設定可能
- **期待値計算**: 設定したバフを反映した期待される火力を自動計算
- **ソート機能**: 期待値、攻撃力、防御力、スロット、会心率でソート可能
- **切れ味補正**: 通常、匠+1、匠+2の切れ味状態を選択可能
- **旋律効果**: 攻撃旋律や会心強化旋律の効果、笛自体が持つ旋律を反映
- **旋律フィルター**: 特定の旋律を持つ狩猟笛を絞り込み

[▶ 狩猟笛比較表を開く](https://morijellyfish.github.io/XXHornTools/huntingHorns)

#### 旋律タイマー

- **複数タイマー管理**: 最大10個のタイマーを同時に管理
- **上限値・延長機能**: 各旋律ごとに初回値と延長値を個別設定
- **テンプレート機能**: 狩猟笛ごとの旋律テンプレートを一括適用
- **キーボード操作**: 0-9キーでタイマーを簡単に操作、タップやクリックでも延長可能
- **笛吹き名人対応**: スキル有無に応じた持続時間の自動調整
- **全旋律延長対応**: 該当旋律を演奏することで全ての旋律タイマーを延長可能
- **自由に追加可能**: 怪力の種、テオノヴァ、怒り時間の管理などにも利用可能

[▶ 旋律タイマーを開く](https://morijellyfish.github.io/XXHornTools/melodyTimer)

## 機能詳細

### 狩猟笛比較表

#### 計算式

期待値は以下の式で計算しています：

```
期待値 = (攻撃力 × 切れ味補正 × 切れ味補正倍率) × 会心補正
```

- **攻撃力**: 各種バフを適用した最終攻撃力
  - すべての加算バフ（力の護符、力の爪、鬼人薬、攻撃スキルなど）を合計した後、乗算バフ（火事場力、不屈、攻撃旋律など）を適用します
- **切れ味補正**: 切れ味の色に応じた物理倍率
- **会心補正**: 会心率に応じたクリティカル期待値（超会心/裏会心を考慮）

## クレジット・免責事項

### ゲームについて

本ツールは、カプコンの「モンスターハンターXX」の攻略をサポートするための非公式ツールです。
モンスターハンターXXはカプコンの著作物です。

### データについて

本ツールで使用しているゲームデータは、モンスターハンターXXのゲーム内情報を参考にしています。

#### 参考にさせていただいたサイト

以下のサイトを参考にさせていただきました。ありがとうございます。

- [MHXX ダメージ計算](https://kuroyonhon.com/mhxx/d/dame.php)
- [MHXG Wiki](http://wiki.mhxg.org/)

### 免責事項

本ツールは非公式のツールであり、カプコンとは一切関係ありません。
データの正確性については保証いたしません。

## 開発者向け情報

### 技術スタック

- **フレームワーク**: [Nuxt 4](https://nuxt.com/)
- **UIライブラリ**: [Nuxt UI](https://ui.nuxt.com/)
- **言語**: TypeScript
- **パッケージマネージャー**: pnpm

### セットアップ

#### 必要な環境

- Node.js（推奨バージョン: 18以上）
- pnpm 10.26.1以上

#### インストール

```bash
# 依存関係のインストール
pnpm install
```

### 開発

#### 開発サーバーの起動

```bash
pnpm dev
```

開発サーバーは `http://localhost:3000` で起動します。

#### ビルド

```bash
# 本番用ビルド
pnpm build

# ビルド結果のプレビュー
pnpm preview
```

#### コード品質

```bash
# ESLintの実行
pnpm lint

# ESLintの自動修正
pnpm lint:fix

# Prettierのフォーマット
pnpm format

# Prettierのチェック
pnpm format:check

# フォーマットとリントの自動修正
pnpm fix

# TypeScriptの型チェック
pnpm typecheck
```

### プロジェクト構造

```
XXHornList/
├── app/
│   ├── components/    # Vueコンポーネント
│   ├── data/          # データファイル
│   ├── pages/          # ページコンポーネント
│   ├── types/         # TypeScript型定義
│   └── utils/         # ユーティリティ関数
├── public/            # 静的ファイル
└── nuxt.config.ts     # Nuxt設定
```

### データの修正

狩猟笛のデータを修正する場合は、`app/data/horns/` ディレクトリ内の該当レア度のファイルを編集してください。

### バフの追加

新しいバフタイプを追加する場合は、`app/types/attackBuff/` ディレクトリ内の該当グループファイルを編集し、計算ロジックを `app/utils/attackBuffCalculate.ts` に追加してください。
